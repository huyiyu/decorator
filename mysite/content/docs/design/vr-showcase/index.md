## 空间印象

### 整体氛围

全屋以**白与木的双色交响**为基调，深棕色地板作为沉稳的底，让空间既有呼吸感又不失分量。白色柜门与木色开放格的交替节奏，像是乐谱上的强弱拍，规律中藏着一点故意的错位——餐边柜的开放格没有居中，而是偏向一侧，那一点不对称让整面墙活了起来。

### 客厅

深色皮质沙发是客厅的视觉锚点，与浅色空间形成克制的重量对比。电视墙做满墙定制柜，电视内嵌，正面望去是一块干净的黑屏悬浮在白木相间的面板之间。沙发旁一盏黑色落地灯，弧线灯罩像一片安静的叶子垂在边几上方。

沙发背景墙目前保持简洁，等待设计师重新规划替代方案（已否决柜子+隐形门）。顶面采用边吊形式，保留原始层高，仅在空调风管和设备的必经之处下吊，空间纵向舒展。

### 厨房与餐厅

岛台是这里的枢纽——白色柜体支撑一块深色木质台面，既是操作台也是餐桌。四把弧形靠背的吧台椅围拢在台边，椅背的曲线柔化了岛台的直线条。台面上随手放着水果盘、插着细枝植物的白瓷花瓶，还有两本平放的书，像是有人刚刚起身离开。

餐边柜做了整面到顶的收纳，白色门板中嵌着一块木色开放格，格内留空，日后可以展示器物。双开门冰箱和嵌入式烤箱都藏进了柜体，冰箱旁留出一列开放式层架，取放高频物品更顺手。

厨房与餐厅之间的移门考虑隐藏式设计，藏入柜体或墙体，敞开时两个空间完全贯通，关上时油烟不扰。

### 主卧

主卧的节奏比公区更慢。大床铺着深灰与棕色相间的床品，床头一侧摆着一盆细叶植物。睡眠区与卫生间之间立了一道白色造型隔断，上面开着三个正圆孔洞——光从孔中透过来时，会在地面投下圆圆的斑点，像月亮的相位。

洗脸台与梳妆台分设两处，梳妆台选用木头材质，给以白木为主调的空间注入一点自然温度。卫生间做了干湿分离，造型隔断重新选型后，两个区域的过渡更加柔和。

### 边角与细节

- **窗边矮柜**：卧室飘窗下方做了一排木色矮柜，开放式的格子里可以收纳书籍或展示小物
- **电动窗帘**：全屋保留，晨起日落一键切换
- **吊柜灯带**：厨房操作面的补光，也是夜间的一盏氛围灯
- **木纹砖**：替代花砖后，地面的视觉节奏更安静，耐脏且易于打理

## 设计亮点回顾

| 空间 | 核心处理 |
|:---|:---|
| 客厅 | 满墙电视柜 + 内嵌电视 + 深色皮质沙发 |
| 厨房 | 中岛台兼餐桌 + 嵌入式厨电 + 开放格餐边柜 |
| 餐厅 | 岛台围合式用餐 + 细枝植物与书籍点缀 |
| 主卧 | 圆孔造型隔断 + 洗脸台与梳妆台分离 + 木材质感 |
| 入户 | 玄关收纳优化，避免进门直视杂乱 |
| 地面 | 全屋木纹砖通铺，暖色奶油系 |
