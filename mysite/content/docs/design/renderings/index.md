## 设计说明

本套方案风格定位为 **奶油中古风**（Cream Mid-century Modern），以下从五个维度总结统一的设计语言：

| 维度 | 设计选择 |
|------|----------|
| **基调** | 奶油色/米白色为空间底色，温暖柔和 |
| **地面** | 浅色木地板通铺，延伸视觉纵深感 |
| **柜体** | 白色柜门为主，深木色开放格/吊柜点缀，圆孔造型增添趣味 |
| **灯光** | 无主灯设计+局部吊灯/壁灯，氛围照明与功能照明兼顾 |
| **软装** | 中古家具元素：弧形线条、木质框架、藤编材质、软包床头 |

**奶油中古风的核心特征：**

- 以奶油色/米白色为空间底色，温暖柔和
- 中古家具元素：弧形线条、木质框架、藤编材质
- 深木色作为点缀色，与白色形成温润对比
- 整体氛围安静、从容、有呼吸感
