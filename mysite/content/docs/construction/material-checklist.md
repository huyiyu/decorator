> 工程地址：阳光城檀映 6#401

本清单按施工顺序分为五个阶段，每一项需在对应工种进场前完成选型、量尺与下单，避免工期延误。

---

## 第一阶段：水电阶段主材定位

水电开工前必须到场或完成定位的设备，直接影响点位、管径及回路设计。

- [ ] 地暖
- [x] 中央空调
- [x] 壁挂空调
- [ ] 厨房空调
- [ ] 负压 / 新风系统
- [ ] 净水系统
- [x] 增压系统
- [ ] 壁挂马桶
- [ ] 壁挂洗衣机
- [x] ~~隐藏花洒~~、水池龙头、智能浴巾架
- [x] 燃热热水器、~~空气能 / 电热水器 / 小厨宝~~
- [x] 烘洗套、冰箱
- [x] 蒸烤箱、洗碗机
- [ ] 集成灶 / 分体灶
- [x] 电视及支架
- [x] 扫地机器人、手持扫地机
- [ ] 投影、音响
- [x] 电脑、打印机
- [ ] 智能系统
- [x] 摄像头

---

## 第二阶段：瓦工阶段主材定位

瓦工进场前需确认材料规格、色差及进场时间，避免等料停工。

- [ ] 防盗门
- [ ] 瓷砖、石材、地漏、挡水条、美缝剂
- [ ] 木地板厚度（如与瓷砖通铺需提前确认收口方案）

---

## 第三阶段：木工阶段主材定位

木工主要负责吊顶基层与窗帘盒，以下项目需提前确认尺寸或预埋件。

- [ ] 晾衣架（是否做隐藏式）
- [ ] 预埋线型灯、轨道灯、射灯（确认开孔尺寸与变压器位置）

---

## 第四阶段：第一道腻子阶段主材进场

腻子施工前，门套与柜体需进场或完成复尺，确保收口缝隙可控。

- [ ] 卧室门、门锁、卫生间门、厨房门
- [ ] 全屋定制柜体、岩板、水槽

---

## 第五阶段：油漆结束主材进场

油漆完工后进入安装收尾，涉及多个商家交叉进场，建议提前一周预约。

- [ ] 木地板、踢脚线
- [ ] 开关插座、灯具、卫浴、五金挂件、暖风机
- [ ] 厨房 / 卫生间集成吊顶、淋浴屏

---
